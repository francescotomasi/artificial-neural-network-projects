# Mars Terrain Segmentation  

## Description  
This project tackles a semantic segmentation task involving 64x128 grayscale images of Mars terrain. The goal is to classify each pixel into one of five terrain classes:  
- 0: Background  
- 1: Soil  
- 2: Bedrock  
- 3: Sand  
- 4: Big Rock
